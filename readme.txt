WooTokenEVM Payment Gateway

== Description ==
Una Pasarela de Pagos para usar Tokens de la Red de Ethereum, Binance y Matic y compatibles con solidity para WooCommerce

== Instalación ==
1. Oblogatorio tener instaado WooCommerce
1. Descarga el complemento WooTokenEVM Payment Gateway (plugin)
2. Instala el complemento
3. Configura las opciones

== Preguntas Frecuentes ==
1. ¿Cómo puedo obtener la dirección de mi contrato?
R: Puedes conseguirlo en etherscan.io / bscscan.com / polygonscan.com
